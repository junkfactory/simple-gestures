SimpleGestures Configuration

SimpleGestures is missing "Access your data for all websites" permission in order to perform mouse gesture commands. Note that no data is actually collected.

Firefox requires users to manually grant this permission to extensions. You can request it by clicking on the button below and then click Allow when Firefox asks you to deny or allow.

After granting permission, you may need to reload existing tabs for gestures to function properly.

See here for more info about this permission.
